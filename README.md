# group-expense-tracker

A simple group expense tracking tool.\
(Final Project – NTU Programming for Business Computing 113-2)

## Run the project

### 1. Clone the project

```bash
git clone https://github.com/Sagehuang/group-expense-tracker.git
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the backend

```bash
cd backend
python app.py
```

### 4. Run the frontend

```bash
cd frontend
python client.py
```

Then you will see the GUI.
